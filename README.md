# Sahara Enterprises Website

Static single-page marketing website for Sahara Enterprises.

## Overview

- One-page brochure site built with plain HTML, CSS, and JavaScript
- Showcases company profile, services, clients, locations, and contact details
- Designed for desktop and mobile with responsive layouts and lightweight animations

## Tech Stack

- HTML5
- CSS3
- Vanilla JavaScript
- Google Fonts

## Project Structure

```text
.
|- index.html   # Complete website markup, styles, and scripts
|- README.md    # Project documentation
```

## Current Features

- Fixed navigation with section links
- Hero section with branded statistics and CTA buttons
- About, Services, Clients, and Locations sections
- Contact section with enquiry form UI
- Scroll reveal effects and navbar shadow on scroll

## Notes

- The website is currently fully self-contained in `index.html`
- The contact form is present as a front-end interface and does not submit to a backend service
- Some fonts and client logos are loaded from external URLs

## Local Usage

Open `index.html` in a browser, or serve the folder with any static file server.

Example:

```bash
python -m http.server 8000
```

Then visit `http://localhost:8000`.

## Deployment

This site can be deployed to any static hosting platform, including:

- GitHub Pages
- Netlify
- Vercel
- Traditional web hosting / cPanel
